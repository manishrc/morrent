morrent
=======

Command-line search and stream movie torrents.

**REQUIREMENTS:**

- [NodeJS](http://nodejs.org/)
- [VLC Media Player](http://www.videolan.org/vlc/index.html)

**INSTALL:**

`npm install -g morrent`

**USAGE:**

`morrent <movie name> [options]`

**OPTIONS:**

```
--hd Plays 1080p
--sd Plays 720p
--3d Plays 3D
```
